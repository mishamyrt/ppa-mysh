## How do I install Mysh?

```sh
curl -s --compressed 'https://mishamyrt.github.io/ppa-mysh/KEY.gpg' | sudo apt-key add -
sudo add-apt-repository 'deb https://mishamyrt.github.io/ppa-mysh ./'
sudo apt update
sudo apt install mysh
```
